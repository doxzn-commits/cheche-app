# 탐색 (홈) — SCR-004

## 화면 설명
체험단 공고 목록 화면. 하단 탭바 첫 번째 탭.
다중 필터(리뷰매체·타입·카테고리·사이트)와 키워드 검색 제공.
공고 클릭 시 SCR-006 체험단 상세로 이동.

## 관련 화면
- SCR-006 체험단 상세
- SCR-006A 인앱 브라우저
- SCR-013 체험단 지도

## 주요 컴포넌트
- SearchBar: 키워드 입력
- FilterChips: 리뷰매체·타입·카테고리·사이트 필터
- CampaignCard: 공고 카드 (플랫폼명·제목·혜택·마감일·인원)
- EmptyState: 검색 결과 없음 (SCR-011)
