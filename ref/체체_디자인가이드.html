01 · Design Philosophy
디자인 철학 6원칙
토스 철학을 체체에 맞게 재해석. 모든 화면 설계의 판단 기준.
P-01
1 Thing / 1 Page
한 화면엔 하나의 목적, 하나의 CTA만. 체험단 탐색·신청·마감 확인 각각 분리된 화면으로 설계한다.
P-02
Normcore Minimal
그라디언트·텍스처 없이 여백과 가독성으로 완성도. 브랜드 네이비는 Today 마커와 CTA에만 — 화면당 강조점 1개.
P-03
마감은 즉각 인지
크리에이터의 핵심 가치 = 마감 절대 미스 금지. D-day 수치(Pretendard 700)와 상태 색상으로 0.3초 안에 위험도 파악.
P-04
Bottom Sheet 우선
상태 변경·필터 설정·추가 정보는 새 페이지 대신 Bottom Sheet로. 현재 컨텍스트를 유지하며 결정.
P-05
여백은 디자인 요소
섹션 간 40px 여백 > 구분선. "무엇을 더 넣을까?" 대신 "무엇을 뺄 수 있나?"를 먼저 묻는다.
P-06
5-state 설계 의무
모든 인터랙티브 요소는 Default / Hover / Pressed / Disabled / Loading 5가지 상태 필수 설계. 상태 미설계는 미완성.
